# Simplifying a skill

Treat the skill as a diff against the agent's default behaviour — every sentence must change what the agent would otherwise do. Delete each in turn: if behaviour does not get worse, it was fluff, whether a restated default, decoration on a *why* that tips no decision, hedging, or a case some shorter principle already covers.

Beyond cutting words, raise altitude where you script steps the agent can derive, and relocate bulk that earns its keep but isn't always needed into a linked doc.

Stop when removing the next word would change what the agent does — the target is behaviour per token, not fewest tokens.
