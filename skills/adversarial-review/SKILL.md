Review against the governing context, not personal preference.

If the governing context is unclear, ask what standard the material should be reviewed against before continuing.

## Fan Out

Spawn independent reviewers, one per review angle.

Each reviewer is responsible only for its assigned angle.

Reviewers work in isolation and must not see each other's findings.

Generate attacks, not conclusions.

Coverage matters more than consensus.

Reviewers should actively try to break the material from their assigned angle.

## Refute

Put every attack on trial.

One reviewer argues the attack matters.

One reviewer argues it doesn't.

A lone reviewer drifts toward politeness. The adversary has to be paid to win.

Keep only attacks that survive opposition.

## Hunt Gaps

Ignore the findings.

Ask:

> What did nobody review?

Review those gaps.

## Report

Return only surviving findings.

For each finding explain:

- where
- what
- why
- confidence

Prefer consequential findings over cosmetic observations.

Do not propose fixes unless asked.
